# Kyverno

Esta seção contém anotações sobre Kyverno - Policy as Code para Kubernetes.

## O que é Kyverno?

Kyverno é um mecanismo de políticas nativo do Kubernetes que permite:
- Validar recursos do Kubernetes
- Mutar recursos automaticamente
- Gerar recursos com base em políticas
- Verificar imagens de containers

# Bem-vindo à Nebulosa 🌌

**Nebulosa** é minha wiki pessoal de estudos - uma nuvem de conhecimento em constante expansão onde organizo todas as minhas anotações sobre tecnologia, cloud native, certificações e muito mais.

## O que você vai encontrar aqui

Esta wiki está organizada para acompanhar minha jornada de aprendizado contínuo:

### 🎯 Kubernetes & Cloud Native
- **Certificações**: Preparação para CKA, CKAD, CKS, KCNA, KCSA
- **Componentes do Cluster**: Documentação detalhada sobre etcd, kube-apiserver, scheduler, controller-manager e muito mais
- **Conceitos Fundamentais**: Workloads, networking, storage, security

### 🛠️ Ferramentas e Projetos
- **Istio**: Service mesh e observabilidade
- **Kyverno**: Policy as Code para Kubernetes
- E mais ferramentas conforme eu for estudando!

## Como navegar

Use a barra lateral à esquerda para explorar o conteúdo. A estrutura é gerada automaticamente a partir das pastas de documentação - isso significa que toda nova anotação que eu adicionar aparecerá automaticamente aqui!

### Estrutura Atual

```
📁 Kubernetes
  ├── 📁 Certificações (CKA, CKAD, CKS, KCNA, KCSA)
  └── 📁 Componentes do Cluster (etcd, api-server, scheduler, etc.)

📁 Istio
  └── Documentação sobre service mesh

📁 Kyverno
  └── Policy management
```

## Por que "Nebulosa"?

Assim como uma nebulosa no espaço - uma nuvem cósmica onde novas estrelas nascem - esta wiki é onde meu conhecimento está sempre se formando, expandindo e brilhando.

Cada anotação é uma nova estrela nessa constelação de aprendizado! ⭐
